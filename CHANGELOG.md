# Changelog

## [0.1.0] - 2026-06-14

### Added

- 初始版本发布
- `.bmsg` 文件语法高亮支持
- 关键字高亮：`schema`, `package`, `enum`, `message`
- 基础类型高亮：`uint32`, `uint64`, `int32`, `int64`, `float32`, `float64`, `string`, `bool`, `bytes`
- 泛型类型高亮：`list<T>`, `map<K, V>`
- i18n 双语注释高亮：`// "中文" | "English"`
- 括号匹配与自动闭合
- 语言配置（注释、折叠等）
